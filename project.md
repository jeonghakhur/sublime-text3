## 프로젝트 만들기
1. 'Project > Add Folder to Project'
2. 'Project > Save as Project'


.sublime-project 파일의 내용은 다음과 같은 json 형식으로 관리됩니다.

```json
{
    "folders":
    [
        {
            "path": "C:\\Users\\Administrator\\work\\my_git_books\\sublime-text3"
        }
    ]
}
```

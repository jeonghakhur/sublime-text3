# 사용자 설정

## Preferences

서브라임 텍스트의 기본설정을 수정합니다. 메뉴의 'Preferences > Settings - User' 를 선택하면 사용자 설정파일이 생성됩니다. 생성된 설정파일은 Json 포맷에 맞추어 작성합니다.

### 폰트 변경
```
    "font_face": "NanumGothicCoding",
    "font_size": 12
```

## Terminal

```json
{
    "terminal": "C:\\Program Files\\Git\\git-bash.exe"
}
```
## Alignment

```Json
{
    "alignment_chars": ["=", "-"],
    "alignment_space_chars": ["=", "-"]
}
```

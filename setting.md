# 사용자 설정

## Preferences

서브라임 텍스트의 기본설정을 수정합니다. 메뉴의 'Preferences > Settings - User' 를 선택하면 사용자 설정파일이 생성됩니다. 생성된 설정파일은 Json 포맷에 맞추어 작성합니다..

```
    "always_show_minimap_viewport": 미니맵에서 현재 위치 표시,
    "bold_folder_labels": 폴더 굵게 표시,
    "caret_extra_bottom": 커서 높이 아래로 키움,
    "caret_extra_top": 커서 높이 위로 키움,
    "caret_extra_width": 커서 너비,
    "caret_style": 커서 스타일,
    "color_scheme": 문자 색상 테마,
    "defalut_line_ending": 줄바꿈 문자 형식 지정(system, windows, unix),
    "default_font_size": 기본 폰트 사이즈
    "draw_minimap_border": 미니맵에 현재 위치 위치 표기에 테두리 표시,
    "draw_white_space": 탭, 공백 시각화하기,
    "ensure_newline_at_eof_on_save": 저장 시 문서 마지막 줄에 빈 줄 없으면 추가
    "fade_fold_buttons": 코드 접기 사용,
    "font_face": 글꼴,
    "font_size": 폰트 사이즈,
    "highlight_line": 현재 줄 강조,
    "highlight_modified_tabs": 변경된 파일 탭 색상 표시,
    "line_padding_bottom": 줄 간격 하단 여백,
    "line_padding_top": 줄 간견 상단 여백,
    "match_selection": 선택된 동일한 단어 하이라이트,
    "preview_on_click": "원 클릭으로 미리보기 실행",
    "rules": 지정한 가로 문자열 위치에 라인 표시,
    "tab_size": 탭 간격,
    "theme": 테마,
    "translate_tabs_to_space": true(스페이스), false(탭),
    "trim_trailing_white_space_on_save": 저장 시 줄끝 공백 제거,
    "word_separators": 단어 선택시 구분자 지정,
    "word_wrap": 자동 줄 바꿈 사용,
```
